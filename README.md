# myWeb\n\nThis project contains `index.html` for the registration card form.\n\n## Deploy to GitHub Pages\n1. Create a new GitHub repository.\n2. Add the repository remote: `git remote add origin https://github.com/<your-username>/<repo-name>.git`\n3. Push to GitHub: `git branch -M main && git push -u origin main`\n4. Enable GitHub Pages in repository settings and select `main` branch.\n\nOnce deployed, your site URL will be `https://<your-username>.github.io/<repo-name>/`.\n
